AI-DRIVEN VOICE ASSISTANT

Veena is an AI-powered, multilingual insurance assistant bot built using Python and large language models (LLMs). It interacts with users through both voice calls and a GUI chat interface to guide them through life insurance policy renewal conversations. The bot intelligently answers questions, handles objections, and provides contextual recommendations in real time. It supports English, Hindi, Marathi, and Gujarati, dynamically switching languages based on user input. Wikipedia integration allows Veena to answer general insurance-related queries. Twilio enables real-time voice calls, allowing Veena to communicate seamlessly over the phone.

Key Highlights :

 
LLM-Powered Conversations: Handles dynamic, context-aware queries.

Twilio Call Integration: Supports real-time voice calls with users.

Multilingual Support: English, Hindi, Marathi, Gujarati with seamless switching.

Voice Interaction: Real-time speech recognition and natural voice responses.

Knowledge Integration: Wikipedia and insurance-related knowledge for FAQs.

GUI-Based Chat: Interactive interface using Tkinter.

Intelligent Objection Handling: Understands and responds to objections meaningfully.
